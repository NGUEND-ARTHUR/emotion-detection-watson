# Final Project

## Emotion Detection Application

This project is a simple **Emotion Detection Web Application** built using:

- Python
- Flask
- Watson NLP Emotion Detection API

---

## 📌 Project Description

The application takes a text input and analyzes the emotions present in the text using a pre-trained Watson NLP model.

It returns the following emotions:

- anger
- disgust
- fear
- joy
- sadness

It also returns the **dominant emotion**.
